This application visualizes the water data of the San Francisco Bay.

You need node.js to build it.
A server is needed to run the application. The main html page is index.html. 
Morever, the folder "css", "data", "fonts", "images" and "js" are required. 

If you have node.js and the modules installed, you can run a local webserver with
    gulp run
on the port 3000.

To install the packages, run
    npm install
in this folder to install all required node.js modules. 

After that, you have the following options:

gulp 
    builds the application

gulp watch
    run the application in development mode

gulp run   
    run a local webserver on port 3000
